# DgenBooks - Premium Expense Tracker

A modern, responsive, and professional expense tracking application built with Next.js, TypeScript, and Firebase.

## Features

- 📊 **Interactive Dashboard** - Real-time expense tracking with beautiful charts
- 💰 **Balance Tracking** - Keep track of who owes what
- 🎨 **Premium UI** - Dark theme with smooth animations and transitions
- 📱 **Fully Responsive** - Optimized for mobile, tablet, and desktop
- ⚡ **Fast & Efficient** - Built with Next.js 15 and Turbopack
- 🔐 **Secure Authentication** - Firebase authentication with email and Google sign-in
- ⌨️ **Keyboard Shortcuts** - Power user features for quick navigation

## Keyboard Shortcuts

- `Ctrl/Cmd + N` - Add new expense
- `Ctrl/Cmd + L` - Go to expense log
- `Ctrl/Cmd + S` - Go to settle up page
- `Esc` - Close dialogs

## Tech Stack

- **Framework:** Next.js 15.5.9
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI
- **Charts:** Recharts
- **Authentication:** Firebase Auth
- **Database:** Firestore
- **AI:** Google Genkit

## Getting Started

### Prerequisites

- Node.js 20 or later
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up Firebase configuration
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:9002](http://localhost:9002)
