# **App Name**: DgenBooks

## Core Features:

- Expense Entry: Record purchases with item name, date, price, and payer.
- Split Logic: Divide costs equally or customize splits among users.
- Repayment Flow: Mark debts as paid and record repayment transactions.
- Dashboard Summary: Display who owes who and net balance for the logged-in user.
- Expense Log: View a detailed history of all expenses with filtering options.
- Settle Up: Settle debts with a matrix or card view and confirmation.
- User Role and Authorization: The tool should allow each of the authorized users to maintain financial information only they created.

## Style Guidelines:

- Primary color: Emerald green (#36B37E) for a professional yet vibrant feel. It will serve as a visible color on the default dark backdrop.
- Background color: Deep slate gray (#0f172a) for a futuristic, clean dark mode aesthetic.
- Accent color: Cyan (#4DD0E1) for actions and highlights to provide visual pop on the dark background.
- Headline font: 'Space Grotesk' (sans-serif) for a techy, scientific feel.
- Body font: 'Inter' (sans-serif) to ensure comfortable reading experience.
- Crisp, neon icons to match the dark theme. Use simple, easily recognizable glyphs for actions.
- Glassmorphism or high-end SaaS style layout. Use cards and tables for content organization. Prominent FAB button on mobile for adding expenses.
- Subtle transition animations for a smooth user experience. Focus on meaningful transitions that enhance usability rather than distract.
